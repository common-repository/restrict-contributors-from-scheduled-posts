Restricts contributors from viewing or editing their posts after the posts are scheduled to publish.

== Description ==

This plugin removes the ability for contributors to view or edit scheduled posts. It works by removing scheduled posts from the contributor's list of posts in the posts admin list. 

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
